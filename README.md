# palette

A working portfolio of Sankha Cooray's UI and interaction design — the visual
side of the network of small web apps that share this domain family.

Deployed at **[palette.sankhacooray.com](https://palette.sankhacooray.com)** via
GitHub Pages.

## Sections

1. **About** — engineer who keeps wandering into design
2. **Selected Work** — live sites: coffeelab, mixlab, ensemble, eo, paynow
3. **House Style** — the recurring decisions across the family
4. **This Site's Palette** — magenta + violet variant of the family theme
